# LAD_2026_JS
Репозиторий с домашними заданиями по JavaScript.

## 📚 О курсе
Школа: Lad-academy  
Направление: Frontend-разработка  

## 📂 Структура репозитория
Каждое домашнее задание находится в отдельной ветке.

Пример:
- homework_1
- homework_2
- homework_3

## 🚀 Как посмотреть задания
1. Открой список веток (branches)
2. Выбери нужную ветку (например, `homework_2`)
3. Посмотри файлы внутри

## 🎯 Цель
Практика JavaScript и закрепление знаний.
